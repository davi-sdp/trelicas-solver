Treliça Solver

Bem-vindo ao Treliça Solver, a ferramenta definitiva para calcular as forças de apoio e forças internas em treliças. Este site é projetado para estudantes de Mecanica dos Sólidos que desejam uma maneira fácil e eficiente de resolver problemas relacionados a treliças.

    Instruções:
  • Use os formulários abaixo para adicionar os nós, barras, forças externas e apoios da sua treliça.
  • As forças externas usam a convenção de sinais de 0° para representar a direção positiva do eixo X e 90° para representar a direção positiva do eixo Y.
  • Após inserir todos os dados, clique em "Calcular Treliça" para obter os resultados.

            Sobre o Treliça Solver

            Esta ferramenta foi desenvolvida para auxiliar estudantes e engenheiros na análise de estruturas isostáticas. O sistema utiliza o método matricial para garantir precisão no cálculo de esforços internos em todas as barras, além de fornecer uma visualização clara das reações de apoio.
